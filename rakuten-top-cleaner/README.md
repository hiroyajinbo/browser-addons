# Rakuten Top Cleaner

Rakuten Top Cleaner は、楽天市場トップページ `https://www.rakuten.co.jp/` の表示を、自分の Firefox 上だけで見やすく整理するためのローカル拡張です。

外部サーバーは使わず、楽天の通信内容も書き換えません。トップページ上にすでに存在する DOM と CSS を、表示整理のために軽く調整します。

## インストール方法

1. Firefox で `about:debugging#/runtime/this-firefox` を開きます。
2. 「一時的なアドオンを読み込む」を押します。
3. このフォルダ内の `manifest.json` を選択します。
4. `https://www.rakuten.co.jp/` を開きます。

一時的なアドオンとして読み込むため、Firefox を再起動した後は必要に応じて読み込み直してください。

## AMO Unlistedで署名して自己配布する手順

1. `manifest.json` の `version` を確認します。初回提出は `1.0.0`、更新時は必ず前回より大きいバージョンへ上げます。
2. ZIP化するときは拡張フォルダ自体ではなく、フォルダの中身を圧縮します。ZIPの直下に `manifest.json`、`assets/`、`src/` が来る形にしてください。
3. ZIPには開発用・確認用の不要ファイルを入れません。通常は `manifest.json`、`README.md`、`assets/`、`src/` だけを含めます。
4. Mozilla Add-ons Developer Hub で新規アドオンを作成し、配布方法に Unlisted を選んでZIPをアップロードします。
5. 審査・署名が完了したら、署名済みXPIをダウンロードします。
6. Firefox の `about:addons` を開き、歯車メニューから「ファイルからアドオンをインストール」を選んで、署名済みXPIを指定します。

この拡張は外部通信、Cookie取得、履歴取得、購入履歴取得を行いません。`browser.storage.local` はポップアップ設定の保存だけに使います。

## 使い方

ツールバーの Rakuten Top Cleaner アイコンを押すと設定ポップアップが開きます。以下の設定を切り替えられます。

- ポイントアップ情報の一覧化
- ショートカットバー表示
- クーポン枠の上部表示
- ショートカットを新しいタブで開く

設定は `browser.storage.local` に保存され、ページ再読み込み後も保持されます。

## できること

- 楽天トップのヘッダー付近に、よく使う楽天サービスへのショートカットバーを追加します。
- ショートカットは初期状態では同じタブで開きます。
- 楽天トップ上にあるポイントアップ情報のスライダー候補を、DOM 上に存在するカードが見えやすい一覧に近い形へ調整します。
- DOM が遅れて生成された場合も `MutationObserver` で再適用します。
- 楽天トップ上に表示済みのクーポン枠が見つかった場合、上部に複製して見やすく表示します。

## やらないこと

- 通信改変はしません。
- 楽天の内部 API 解析はしません。
- キャンペーンページを裏で自動巡回しません。
- クーポン獲得ボタンを大量自動クリックしません。
- ログイン情報、Cookie、個人情報、購入履歴を取得・保存しません。
- 広告ブロック目的の拡張ではありません。

## 既知の制限

- 楽天側の DOM やクラス名が変わると、一部の整理機能が動かなくなる可能性があります。
- ポイントアップ情報はトップページ DOM 上に存在するカードだけを対象にします。
- クーポン枠はトップページ上に表示されているものだけを扱います。
- クーポンの上部表示は複製表示です。ユーザーが明示的にクリックした場合のみ、表示中のリンクやボタンが動作します。

## 動作確認項目

- Firefox で一時的なアドオンとして読み込める
- `https://www.rakuten.co.jp/` でのみ動作する
- 他サイトでは何もしない
- ショートカットバーが表示される
- 各ショートカットが正しい URL を開く
- 設定でショートカットバーを OFF にできる
- 設定で新しいタブ/同じタブを切り替えられる
- ポイントアップ情報が存在する場合、スライダー形式ではなく一覧に近い形で見える
- ポイントアップ情報が見つからない場合でもエラーにならない
- クーポン枠が存在する場合、上部に見やすく表示される
- クーポン枠が見つからない場合でもエラーにならない
- ページ再読み込み後も設定が保持される
