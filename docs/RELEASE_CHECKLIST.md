# リリースチェックリスト

公開、署名、配布前に確認する項目です。

## 共通

- [ ] `manifest.json` の `name`、`description`、`version` が正しい
- [ ] 非公式ツールであることを説明している
- [ ] 不要な権限を要求していない
- [ ] APIトークンや秘密情報がコミットされていない
- [ ] 巨大な生成ファイルがコミットされていない
- [ ] READMEが現状に合っている
- [ ] `PRIVACY.md` が現状に合っている
- [ ] `SECURITY.md` が現状に合っている
- [ ] `SUPPORT.md` が現状に合っている
- [ ] アイコン、スクリーンショット、説明文が最新

## Fastmail系

- [ ] APIトークンの設定手順が分かる
- [ ] APIトークン削除または再設定ができる
- [ ] 接続エラーが分かりやすく表示される
- [ ] 必要なFastmail API権限が説明されている
- [ ] メールHTMLをサニタイズしている
- [ ] 外部画像の扱いを説明している
- [ ] リンクが安全に新しいタブで開く
- [ ] 既読化など破壊的操作に問題がない

## 動作確認

- [ ] Firefoxで一時読み込みできる
- [ ] popupが開く
- [ ] optionsが開く
- [ ] 設定保存ができる
- [ ] 拡張機能の再読み込み後も設定が復元される
- [ ] 通知、バッジ、アイコン状態が期待通り
- [ ] 通知対象フォルダを全解除したとき、バッジと通知が0件扱いになる
- [ ] 通知対象外フォルダの未読メールがポップアップでは確認できる
- [ ] HTML表示ON/OFF、外部画像ON/OFFが期待通り
- [ ] 通知・表示、通知対象フォルダの変更が自動保存される
- [ ] エラー時に壊れた表示にならない

## 配布

- [ ] XPIやZIPに不要ファイルを含めていない
- [ ] `.git/`、ローカルメモ、巨大データを含めていない
- [ ] AMOまたはChrome Web Storeの説明文に権限理由を書いた
- [ ] プライバシーポリシーへのリンクを用意した
- [ ] サポート方針を明記した

## リリース後

- [ ] タグを作成した
- [ ] CHANGELOGを更新した
- [ ] 自分の環境でインストール確認した
- [ ] Issueや問い合わせが来ても対応範囲を超えないようにする
