# セキュリティ方針

このリポジトリの拡張機能は、メール、APIトークン、Webページ内容など、慎重に扱うべき情報に触れる可能性があります。

## セキュリティ上の基本方針

- APIトークンを開発者のサーバーへ送信しません。
- リモートコードを読み込みません。
- メールHTMLをそのまま信頼しません。
- 不要なブラウザ権限を要求しません。
- 機能に必要な外部通信のみを行います。

## HTMLメール表示

メールHTMLを表示する拡張機能では、以下のような危険な要素を削除または無効化します。

- `script`
- `iframe`
- `object`
- `embed`
- `form`
- `input`
- `button`
- `style`
- `link`
- イベント属性 `onclick` など
- `javascript:` URL

リンクは新しいタブで開き、`noopener noreferrer` を付与します。

## APIトークン

APIトークンはブラウザのローカルストレージに保存されます。READMEと設定画面では、ユーザーが必要な権限だけを持つAPIトークンを作成できるように、権限の目安を明記します。

Fastmail連携機能では、ユーザーが設定したAPIトークンをFastmail APIへ送信します。開発者のサーバーや解析サービスへは送信しません。

## 外部画像

メールHTML内の外部画像を表示すると、画像配信元へアクセスが発生します。Fastmail Unread Notifierでは、外部画像の表示を設定でON/OFFできます。公開・配布版ではOFF運用を推奨します。

## 脆弱性の報告

このリポジトリは個人開発のため、対応を保証するものではありません。ただし、APIトークン漏えい、任意コード実行、メールHTMLの危険な実行、過剰なデータ送信などの問題を見つけた場合は、GitHub IssuesまたはGitHubの連絡手段から報告してください。

報告時には、可能な範囲で以下を含めてください。

- 対象の拡張機能
- 再現手順
- 影響範囲
- 使用ブラウザとバージョン
- 期待される修正方針

## サポート範囲

このプロジェクトは非公式の個人開発プロジェクトです。対象サービスの提供元へ問い合わせないでください。
