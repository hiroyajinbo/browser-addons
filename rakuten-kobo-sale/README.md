# Rakuten Kobo Sale Exporter

楽天Koboセールページで「もっと見る」を自動クリックし、展開完了後に書籍タイトル・価格・商品URLをCSVで保存するローカル用ブラウザ拡張です。

## 使い方

1. ブラウザの拡張機能管理画面で、この `rakuten-kobo-sale` フォルダを一時的な拡張機能として読み込みます。
2. 楽天Koboの対象セールページを開きます。
3. ツールバーの `Kobo Sale Exporter` を開き、`展開してCSV保存` を押します。
4. ページ右下のパネルが進捗を表示します。
5. 「もっと見る」がなくなると、CSVが自動でダウンロードされます。

## 出力

CSV列は以下です。

- `書籍タイトル`
- `価格`
- `商品URL`

商品URLは `https://books.rakuten.co.jp/rk/...` の商品ページURLに正規化しています。

## 補足

既に全件表示済みのページでは `現在表示分をCSV保存` を押すと、その場の表示内容だけを保存できます。

## AMO Unlistedで署名して自己配布する手順

1. `manifest.json` の `version` を確認します。初回提出は `1.0.0`、更新時は必ず前回より大きいバージョンへ上げます。
2. ZIP化するときは拡張フォルダ自体ではなく、フォルダの中身を圧縮します。ZIPの直下に `manifest.json`、`assets/`、`src/` が来る形にしてください。
3. ZIPには開発用・確認用の不要ファイルを入れません。通常は `manifest.json`、`README.md`、`assets/`、`src/` だけを含めます。
4. Mozilla Add-ons Developer Hub で新規アドオンを作成し、配布方法に Unlisted を選んでZIPをアップロードします。
5. 審査・署名が完了したら、署名済みXPIをダウンロードします。
6. Firefox の `about:addons` を開き、歯車メニューから「ファイルからアドオンをインストール」を選んで、署名済みXPIを指定します。

## 権限と通信

- 追加の `permissions` は要求しません。
- 対象ページは `https://books.rakuten.co.jp/*` のコンテンツスクリプトだけです。
- 外部通信、Cookie取得、履歴取得、購入履歴取得は行いません。
